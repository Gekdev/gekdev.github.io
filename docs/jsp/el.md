## Expression Language, EL

### What is Expression Language?

**${와 }사이에 정해진 문법을 따르는 식을 입력**

JSP 스크립트 코드를 사용하는 것보다 표현 언어를 사용하는 것이 코드를 간결하고 이해하기 좋게 만들어주기 때문에 특별한 이유가 없는 한 표현 언어를 주로 사용함
