브라우저객체모델

브라우저객체모델(BOM, Browser Object Model)은 웹브라우저와 관련된 객체의 집합을 의미한다.
대표적인 BOM으로는 window, location, navigator, history, screen, document객체가 있다.
간단하게 문서객체모델(DOM, Document Object Model)이라 통합해서 부르기도 한다.
