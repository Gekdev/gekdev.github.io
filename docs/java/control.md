## Control Flow Statements

**제어문**

**원하는 결과를 얻기 위해서는 프로그램의 순차적인 흐름을 제어할 때 사용하는 명령문**

&#9656; 조건문, 반복문 등이 있음

&#9656; 제어문에 속하는 명령문들은 중괄호({})에 둘려싸여 있고, 중괄호 영역을 block이라고 함
