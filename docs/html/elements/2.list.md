## List basic 

### What is list?

**목차를 만드는 태그**

### Features of List

&#9656; Can be nested

&#9656; 다른 요소들을 포함할 수 있음

&#9656; 웹 접근성을 위해 네비게이션처럼 동일항목이 2개이상 중복된다면 거의 다 쓰인다 

### Basic form of list

```html
* Ordered list
<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol> 

* Unordered list
<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

* Description List
<dl>
  <dt>Coffee</dt>
  <dd>- black hot drink</dd>
  <dt>Milk</dt>
  <dd>- white cold drink</dd>
</dl>
```

## List elements

### &#60;ol&#62; 

**Ordered list**

* type attribute : define the numbering type (1, A, a, I, i)

* start attribute : Control List Counting

    ```html
    <ol start="50">
      <li>Coffee</li>
      <li>Tea</li>
      <li>Milk</li>
    </ol>

    <ol type="I" start="50">
      <li>Coffee</li>
      <li>Tea</li>
      <li>Milk</li>
    </ol>
    ```

### &#60;ul&#62; 

**Unordered list**

```html
<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
```

### &#60;dl&#62; 

**Description Lists**

* &#60;dl&#62; : description list

* &#60;dt&#62; : defines a term 

* &#60;dd&#62; : describes the term

```html
<dl>
  <dt>Coffee</dt>
  <dd>- black hot drink</dd>
  <dt>Milk</dt>
  <dd>- white cold drink</dd>
</dl>
```
