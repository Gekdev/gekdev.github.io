## List elements

### &#60;ol&#62; 

**Ordered list**

```html
<ol>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
</ol>
```

* type attribute : define the numbering type (1, A, a, I, i)

* start attribute : Control List Counting

    ```html
    <ol start="50">
      <li>Coffee</li>
      <li>Tea</li>
      <li>Milk</li>
    </ol>

    <ol type="I" start="50">
      <li>Coffee</li>
      <li>Tea</li>
      <li>Milk</li>
    </ol>
    ```

### &#60;ul&#62; 

**Unordered list**

```html
<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
```

### &#60;dl&#62; 

**Description Lists**

* &#60;dl&#62; : description list

* &#60;dt&#62; : defines a term 

* &#60;dd&#62; : describes the term

```html
<dl>
  <dt>Coffee</dt>
  <dd>- black hot drink</dd>
  <dt>Milk</dt>
  <dd>- white cold drink</dd>
</dl>
```
