### Resize Property

**리사이즈 되어야하는지 결정하는 속성**

* value 
    
    &#9656; horizontal : 가로로 리사이징
    
    &#9656; vertical : 세로로 리사이징
    
    &#9656; both : 둘다 리사이징
    
    &#9656; none : 리사이징 안되게
    
```css
div {
  resize: horizontal;
  overflow: auto;
}
```
